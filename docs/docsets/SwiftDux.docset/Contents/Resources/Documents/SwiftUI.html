  • MappedState

    Retrieves a mapping of the application state from thje environment and provides it to a property in a SwiftUI view. Use the mapState(from:for:_:) method.

    struct MyView : View {

  @MappedState var todoList: TodoList

}

  • Dispatcher

    Retrieves the dispatcher for the given state from the environment and maps it to a property in a SwiftUI view.

    struct MyView : View {

  @Dispatcher var dispatch: Dispatch

  func login() {
    dispatch(UserActions.login(...))
  }

}
