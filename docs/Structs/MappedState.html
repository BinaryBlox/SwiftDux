MappedState

@propertyDelegate
public struct MappedState<State> : DynamicViewProperty where State : Decodable, State : Encodable

Retrieves a mapping of the application state from the environment and provides it to a property in a SwiftUI view. Use the mapState(updateOn:_:) method.

struct MyView : View {

  @MappedState var todoList: TodoList
