StateMapper

public struct StateMapper<KindOfAction, Superstate, Substate> : ViewModifier where KindOfAction : Action, Superstate : Decodable, Superstate : Encodable, Substate : Decodable, Substate : Encodable

Maps a superstate to a substate. It updates views using a passed in change publisher.

This should not be used diectly, but through View.mapState(updateOn:_:).
